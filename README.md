# Commentor
Only takes the comments in the URLs. Only this =)

# Installation
```
$ git clone
$ cd Commentor
$ pip3 intall -m requirements.txt
```
# Usage

For a single URL:

```
EXAMPLE:
python3 main.py --url https://google.com
```

For a list of URLs:

```
EXAMPLE:
python3 main.py --file list-with-urls.txt
```
